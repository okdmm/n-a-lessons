## (2)
$ f(x) = x^3 + x^2 - 5x + 3 $
とする。
$ f(x) = 0 $の解を求める。
$ f(1) = 0, f'(1) = 0 $ より $ f(x) $を$ (x-1)^2$ で割れば
$$ f(x) = (x-1)^2(x+3) $$
よって $f(x) = 0$の解は$$ x = -3, x = 1(重解)$$である。
